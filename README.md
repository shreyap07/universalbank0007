# Universal Bank AI Marketing Dashboard

This Streamlit application helps marketing teams identify customers most likely to accept a personal loan.

## Features

• Descriptive analytics dashboard with interactive charts  
• Decision Tree, Random Forest, Gradient Boosting models  
• Model comparison table (Accuracy, Precision, Recall, F1)  
• Combined ROC curve for all models  
• Confusion matrices with counts and percentages  
• Upload new customer data and download predictions  

## Run Locally

pip install -r requirements.txt
streamlit run app.py

## Deploy

Upload the project to GitHub and deploy using Streamlit Cloud.
